# macOS-Themed Portfolio Website

A modern, interactive portfolio website inspired by macOS design, built with React, GSAP animations, and Tailwind CSS. This project features a sleek, responsive interface that mimics the macOS experience while showcasing your work and skills.

## Live Demo

Visit the live site: [https://lalitmahato.xyz](https://lalitmahato.xyz)

## Tutorial Credit

This project was created by following the tutorial from the **JavaScript Mastery** YouTube channel. All initial assets and design concepts are credited to JavaScript Mastery.

- Tutorial: [JavaScript Mastery YouTube Channel](https://www.youtube.com/@javascriptmastery)

## Features

- macOS-inspired user interface
- Smooth animations powered by GSAP
- Fully responsive design with Tailwind CSS
- Built with React for optimal performance
- Interactive UI components

## Technologies Used

- React
- GSAP (GreenSock Animation Platform)
- Tailwind CSS
- JavaScript

## Getting Started

```bash
# Clone the repository
git clone https://github.com/mahatolalit/my-site.git

# Navigate to project directory
cd my-site

# Install dependencies
npm install

# Start development server
npm start
```
